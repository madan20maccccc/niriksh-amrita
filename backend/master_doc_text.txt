4. The 4 Innovations (What Makes This Publishable)
These innovations are NOT in standard EWS systems. They are your original contribution and make this project research-grade.
Innovation 1 — Predictive Deterioration
Standard systems only detect current problems. We predict WHEN a patient will cross a danger threshold if the trend continues.
Take last 3 shift readings for any vital sign
Fit a linear trend using NumPy (5 lines of code)
Output: 'At current rate, SBP will cross 180 in approximately 1-2 more shifts'
Backend file: backend/app/innovations/predictor.py
Frontend: New card on patient page showing predicted trend with a simple arrow chart
Innovation 2 — RAG Clinical Assistant
Nurses can ask questions about a specific patient and get answers from that patient's own data only.
Nurse types: 'Why is Bed 1204 flagged orange?' or 'What should I watch tonight?'
System retrieves full vital history + alert log for that patient
Gemini API answers using ONLY that patient's data — no hallucination possible
Reuses our existing RAG chatbot code (Gemini + context injection)
Backend file: backend/app/innovations/rag_assistant.py
Frontend: Chat box panel on patient detail page
Innovation 3 — Explainability Panel
Every alert shows EXACTLY which rules fired — not just a color badge.
Example output:
Orange Alert triggered because:
  -> SBP rose 142->158->176 (+10% per shift)  [Trend Rule]
  -> Glucose above 180 for 3 shifts            [Persistent Rule]
  -> Patient is diabetic + post-op Day 2       [Risk Factor]
Connects to our Grad-CAM explainability research background
Backend: rule_engine returns which rules fired as a list
Frontend: Expandable 'Why was this triggered?' section on every alert card
Innovation 4 — Multilingual SBAR
Generate shift handover summaries in Tamil, Hindi, or English based on nurse preference.
Gemini generates SBAR in English first
Pass through Bhashini API for translation (used in our hackathon project already)
Nurse selects preferred language in SBAR page
Directly addresses India-specific gap — no existing EWS system does this
Backend file: backend/app/innovations/multilingual.py
Frontend: Language toggle buttons on SBAR page (English / Tamil / Hindi)
